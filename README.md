# a1_mpc_raisim
 A control example code of A1 robot

## System requirements:
Ubuntu 18.04, ROS Melodic

## Dependency
install raisim according to the instruction  
https://raisim.com/sections/Installation.html

## Build
``` 
cd {your workspace}  
mkdir build  
cmake ..  
make 
```  

## Running
```
cd bin  
./whole_body_controller  
```
